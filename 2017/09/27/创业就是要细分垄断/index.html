《创业就是要细分垄断》读书笔记

发表于 2017-09-27 | 分类于 创业 | 阅读次数:

作者：李开复、汪华、傅盛

#　序言

  • 创业存在一条重要规律——幂定律，存在于融资、战略、人才各方面。了解这一定律，才有机会在创业一开始就成为垄断者，从非常小的关注点做起，并在垄断形成后快速扩张，做大一个潜在的市场，成就一家伟大的公司。

  • 创业者要做“天下大乱”中的“农民起义军”，趁势而起，抓住新垄断产生的时机，建立壁垒，快速增长、快速扩张，直至垄断整个市场，实现从小垄断到大垄断的完美目标。

  • 创业公司要最求差异化，依靠优秀的战略、严密的执行取胜。

#　第一章 互联网创业基本法则：成为细分市场垄断者

创业是一将功成万骨枯，平均值毫无意义

  • 正太分布对创业者做的事情没有意义，没有什么平均值，只有极少数是特别特别伟大的，其他的都可以忽略，这就是幂定律。

资本追逐幂定律，VC赌的是超级冠军

  • 资本：追求幂定律；VC的模式很简单，找到最好的idea投一次还不够，还要投第二次、第三次；千万不要贪图很多个VC，多有多的好处，但是如果有一个VC非常看中你的项目，要投两三轮的话，一定要给他投资的机会。

垄断才是互联网创业的核心

  • 战略：符合幂定律；对于一个公司，细分领域里做不到第一名，就什么都不是。

  • 在互联网领域，垄断才是核心，而不是竞争。创业公司要找到一个细分市场，然后垄断它，再逐渐地扩大。

  • 互联网创业不是做资源，而是做技术，技术不断有后浪推前浪。

  • 互联网领域要垄断怎么做：第一，先要找到一个非常细分的领域，成为这个领域的垄断者。要专注。要非常深度地去分析， 政策也好，资源也好，变化也好。第二，找到小垄断领域之后，就要让量涨起来。需要以二次方的速度成长。垄断者要在四个方面具有优势：技术、网络效应、品牌和规模经济。

  • 垄断五部曲：

graph LR;
　　分析趋势、变化、政策–>巩固：用生态系统、合作形成护城河–>寻找小垄断、滩头真滴–>增长达到垄断+打造优势维护垄断地位–>扩张+复制：动用资本力量，横向/纵向高速扩张;

顶尖人才决定企业成败

  • 人才：符合幂定律。

  • 创业者首先要成为这样厉害的人，其次要去挖这样的人为己所用。创业者在初期甚至要花一半的工作时间去挖人，一定要把最棒的人留在自己身边，如果比自己还厉害就更好了。

  • 你挖人才的时候不能只靠钱，虽然钱、股份很重要，你还要能够讲出公司的愿景，让他们知道，在你的公司里他们可以扮演伟大的角色，甚至让公司变得更伟大。

  • CEO应该花30%以上的时间在人才上：超频目标不是超过最低期望，而是提升平均水平；清晰描述公司愿景，让每个人知道自己的关键性；打造公司品牌形象，对针对性人物有吸引力。

  • 公司小的时候：最早的一批人定义了公司文化；对于超级员工：不要吝啬股份。

  • 公司大一点以后：深度认识你的核心员工（10%）；当心经理“填满办公室”的心态；善待离职员工，未来或许会回来+为了公司形象。

  • 管理聪明人的而方法：你想怎么被管理，就怎么管理人。放权，不要做微观管理，让员工成为自己的CEO；用OKR（目标与关键成果法），让员工自定义工作目标，并透明分享。
